# Create Sinatra
Is a simple class to generate the necessary files to begin with a commonly Sinatra app

## How to use it?
    $ ruby create_sinatra.rb <name_app>

## After create it
Go to the name_app:
    $ cd <name_app>

Run bundle:
    $ bundle install

Run server:
    $ shotgun

After this *ENJOY IT* `localhost:9393`
